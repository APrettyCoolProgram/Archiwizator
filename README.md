# ABOUT
Archiwizator is a Graphical User Interface (GUI) front-end for [7-Zip](https://www.7-zip.org/).

**Archiwizator v1.0 is pretty rough! It works, but it's a bit wonky. Use at your own risk!** 

### Features
* Batch archiving
* Datestamp postfixes
